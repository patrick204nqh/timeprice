timeprice offline historical inflation & FX for Ruby
GitHub RubyGems

What was $100 in 1990 worth today?

timeprice ships every byte of historical CPI and FX data inside the gem. No API keys, no rate limits, no network. Try it below — the calculator runs the actual gem in your browser. 

gem install timeprice
